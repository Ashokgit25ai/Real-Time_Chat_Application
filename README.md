# Real-Time Chat Application

A real-time chat application developed during internship training using
React, Node.js, and Socket.io.

## Features
- User login with username and room
- Real-time messaging using WebSockets
- Room-based chat system
- Responsive UI

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Real-time: Socket.io

## Note
This project was developed as part of guided internship training.
I worked on understanding the architecture, frontend components,
and real-time communication flow implemented using Socket.io.
